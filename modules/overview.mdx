title: 'Platform Modules Overview'
---

Ratel Security is organized into **8 user-facing modules**. Each module is accessible via REST API and the compliance dashboard.

### Roles & Permissions

| Role | Access Level |
|------|-------------|
| `SUPER_ADMIN` | Full platform access — manage tenants, users, all modules |
| `BANK_ADMIN` | Full access within their tenant — users, rules, cases, reports |
| `COMPLIANCE_OFFICER` | Case management, reporting, transaction review, rule management |
| `ANALYST` | Read access to transactions, cases, and reports. Can create cases and add notes. |
| `AUDITOR` | Read-only access to audit logs, transactions, and reports |
