## Overview

Verifow sends transactional email alerts to keep compliance teams informed of critical events without requiring constant dashboard monitoring.

## Notification Events

| Event                             | Recipients                  | Content                                                            |
| --------------------------------- | --------------------------- | ------------------------------------------------------------------ |
| Transaction BLOCK/ESCALATE/REVIEW | Compliance Officers, Admins | Risk level, score, amount, sender, triggered rules, dashboard link |
| New user created                  | The new user                | Welcome email with temporary login credentials                     |
| Case assigned                     | Assigned officer            | Case title, priority, due date, direct link                        |
| Report approved                   | Report creator              | Confirmation with XML download link                                |
| KYC status change                 | Applicant                   | Verification progress updates                                      |

## Configuration

No configuration is required for core alerts. Email addresses are pulled from user profiles.
